# 韓国語初級単語 Ankiデッキ用 CSV 生成 — 作業記録

## 目的

한국어기초사전（韓国語基礎辞典）のオープンAPIを使って、  
**初級 (level1) の全単語を韓国語→日本語のAnkiデッキ用CSVとして出力する**。

要件：
- 同形語（同じ綴りで異なる意味を持つ語）は **1レコードにまとめる**
- 品詞 (POS) を**日本語に翻訳**する（例: `명사` → `名詞`）

---

## データソース

- **API**: [한국어기초사전 Open API](https://krdict.korean.go.kr/api/search)
  - 初級 (`level1`) の全単語：重複前 **2,501件**、グループ化後 **~1,143件**
  - 日本語翻訳付き取得: `translated=y&trans_lang=2`

---

## 品詞マッピング

| 韓国語 | 日本語 |
|--------|--------|
| 명사 | 名詞 |
| 동사 | 動詞 |
| 형용사 | 形容詞 |
| 부사 | 副詞 |
| 감탄사 | 感動詞 |
| 대명사 | 代名詞 |
| 관형사 | 冠形詞 |
| 수사 | 数詞 |
| 의존 명사 | 依存名詞 |
| 조사 | 助詞 |
| 접사 | 接辞 |
| 보조 동사 | 補助動詞 |
| 보조 형용사 | 補助形容詞 |
| 어미 | 語尾 |
| 품사 없음 | 品詞なし |
| （空欄：複合名詞句） | 名詞（句） |

複数品詞を持つ語は `/` 区切りで結合（例: `감탄사/대명사` → `感動詞/代名詞`）。

## スクリプト

### `scripts/fetch_level1_anki.py` ⭐ メイン

APIから全件取得してAnki用CSVを生成する本命スクリプト。

```bash
python3 scripts/fetch_level1_anki.py
```

**動作：**
1. ハングル全音節（가〜힣）を先頭文字として順次検索
2. 各音節に対してページング（num=100, start=1〜900）で全件取得
3. `target_code` で重複排除
4. 同じ `word` をグループ化（同形語を1レコードに統合）
5. 品詞を日本語に翻訳
6. `data/level1_anki_from_api.csv` に保存

**途中再開：**  
`data/level1_api_progress.jsonl`（完了済みプレフィックス）と  
`data/level1_api_raw.jsonl`（取得済みデータ）が保存されるため、  
中断後に再実行すれば続きから再開できる。

**出力CSVカラム：**

| カラム | 内容 |
|--------|------|
| 韓国語 | 見出し語 |
| 品詞 | 日本語訳済み品詞 |
| 日本語訳 | 見出し語の日本語翻訳（複数義は ` \| ` で結合） |
| 定義 | 日本語での意味説明（複数義は ` \| ` で結合） |
| target_codes | APIのコード（参照用） |

---

### `scripts/create_anki_level1.py`

既存の `data/level1_grouped.csv`（1,143件）から  
品詞翻訳のみ行ってAnki用CSVを生成するスクリプト。  
APIが使えない環境でのフォールバックとして使用。

```bash
python3 scripts/create_anki_level1.py
# → data/level1_anki_jp.csv を生成
```

## Ankiへのインポート方法

1. Ankiを開く → `ファイル` → `インポート`
2. `level1_anki_from_api.csv`（または `level1_anki_jp.csv`）を選択
3. フィールドのマッピングを設定：
   - フィールド1 → 表面（韓国語）
   - フィールド2 → 品詞
   - フィールド3 → 裏面（日本語訳）
   - フィールド4 → 定義（オプション）
4. デリミタ: カンマ
5. インポート実行
